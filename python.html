1.1 Create variables

Create variables

Variables are used to store values.
To create a variable the variable must be named.
Because the names must not contain spaces, these are replaced by snake-cases.
With the '=' you can assign a text value to the variable.
Text values ​​are surrounded by double quotes.

game_1 = "Minecraft"


1.2 Variable types

Variable types

Variables can also store other types of values, not just text values.
Other types are (decimal) numbers and true/false values.
These values ​​are stored without double quotes.

score = 42

pi = 3.14

power_on = True


1.3 Print data

Print data

With the print() function you can print data in the console.
Data can be variables but also values ​​that can be stored in a variable.

score = 42

print("Your Score is:")

print(score)
